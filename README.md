
# nx-medusa-plugin

## Install

Install with:

```sh

yarn -D add nx-medusa

```

## Generate a plugin

Run `nx g nx-medusa:plugin my-plugin` to generate a plugin.

## Generate a module

Run `nx g nx-medusa:module`

1. `name` is the name of the module.
2. `pluginName` name of the plugin where the module should be in.
